Now look, I love Cooking. It makes creating content easy and very user friendly. However, recently I had to create a very large image gallery (400+ images) and I really didn't want to upload all the images to my WordPress site.

So I starting thinking of how I could create it without buy many stuff and how to automate it as much as possible. Here is the solution I came up with.


First thing is first I created a directory of all the images called pictures (rocket science I know). I placed it in a directory that I wanted to host the gallery at (let's call it "gallery" shall we?), I also created an file inside the gallery directory.

There was no way I was going to write html for each image inside the pictures directory so I use Chef to automatically handle it for me:

  1. Bumbu Dapur
  2. Bumbu Sayur
  3. Bumbu Catur
  4. Bumbu Kapur

Cara Memasak

  1. Masukkan bumbu dapur  [*]
  2. Campur dengan bumbu Sayur
  3. Sambil diparut dengan bumbu catur
  4. Dihidangkan dengan bumbu kapur
